# Flask でつくる LINE Chatbot

Flask で作成した LINE ChatBot のバックエンド API．LINE Developers のコンソールから作成した Messaging API からこのバックエンド API を叩くことでレスポンスを返す．

### 開発環境

- Macbook : macOS Monterey (12.3.1)
- Python : 3.8.2
- Flask : 1.1.2
